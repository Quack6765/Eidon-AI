Key Issues Identified

1. Desktop Horizontal Space Underused
Message column capped at 1024px (max-w-5xl) on wide screens. Bubbles further limited to 82% of that (~840px). On a 1920px screen, ~55% of width is empty.
2. Tool Call Cards Too Wide
Tool calls stretch to fill the full column width instead of being compact. Content (JSON, URLs) is shown truncated rather than collapsible.
3. Conversation Flash on Complete
syncConversationState() re-fetches all messages after stream ends, causing a visible flash/re-render of the entire conversation.
4. Mobile Layout Not Optimized
Only 2% width difference between mobile (84%) and desktop (82%). No fluid typography or viewport-based sizing. Minimal responsive differentiation.
