Key Design Decisions

Sidebar Behavior

  • Same width as chat sidebar (280px) — no layout shift
  • Back arrow at top-left replaces logo link
  • "Settings" title below the back arrow
  • Colored icon badges match each section's accent color (purple, sky, amber, etc.)
  • Active item has a subtle tinted background + border matching its accent color
  • Hover state on inactive items shows a subtle dark highlight
  • Smooth transition when switching between chat and settings (framer-motion)

Mobile Behavior

  • Same slide-in overlay as current chat sidebar
  • Settings nav items replace conversation list content
  • Back arrow dismisses sidebar and returns to chat
  • Selecting a section navigates to it and closes the sidebar
